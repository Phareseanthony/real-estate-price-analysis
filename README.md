Real Estate Price Analysis Project
 Overview

This project focuses on data preprocessing and exploratory data analysis (EDA) of a real estate dataset using Python. The goal is to understand the key factors that influence house prices and prepare the data for machine learning models.

📊 What I Did
Loaded and explored dataset using Pandas
Checked data types and missing values
Encoded categorical variables
Scaled numerical features using StandardScaler
Detected and removed outliers using IQR method
Performed correlation analysis
Visualized data using Seaborn and Matplotlib
📈 Key Insights
House price is strongly influenced by:
Area
Number of bedrooms
Bathrooms
Stories
Outlier removal improved data consistency
Correlation heatmap helped identify important features
